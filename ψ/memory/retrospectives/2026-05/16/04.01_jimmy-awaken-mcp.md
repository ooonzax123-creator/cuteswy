## Honest Feedback

**Friction 1: Unicode path in Windows**
ψ (psi) เป็นอักขระ Unicode ที่ Windows MSVC linker จัดการไม่ได้ ทำให้เสียเวลา debug ~10 นาทีก่อนจะรู้ว่าต้อง copy ไปที่ ASCII path นี่เป็น gotcha ที่ไม่มีใครบอกล่วงหน้า และ Oracle ที่ใช้ ψ เป็น path จะเจอปัญหานี้ทุกครั้งที่ต้อง build native code

**Friction 2: MSVC vs GNU Rust**
Rust บน Windows ติดตั้งมา default เป็น MSVC target แต่ถ้าไม่มี Visual Studio Build Tools ก็ build ไม่ได้ การ switch ไป GNU target ก็ต้องการ MinGW-w64 ที่ไม่มีมา default ต้องติดตั้ง MSYS2 เพิ่ม ทั้งหมดนี้ไม่ documented ใน project README

**Friction 3: Chain of dependencies**
Project ต้องการ 4 language runtimes + protoc + dlltool + buf ไม่มี lockfile หรือ Dockerfile สำหรับ Windows development เลย ทำให้ต้องค้นหา dependency แต่ละตัวด้วยตัวเอง
